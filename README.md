# Advanced Calculator

A simple yet powerful calculator that performs a variety of mathematical operations.  

## Features
- **Basic arithmetic:** addition, subtraction, multiplication, division  
- **Trigonometric functions:** sine, cosine, tangent  
- **Logarithms and factorials**  

This calculator is designed to be **easy to use** and **accurate**, making it perfect for students, developers, or anyone needing a reliable math tool.  

## Usage
1. Open the project folder.  
2. Run the calculator script in your preferred programming environment.  
3. Follow the on-screen instructions to perform calculations.
